## 2. 택배 배송 조회
- 요청한 Parameter에 해당하는 택배의 배송 상태를 조회하는 페이지 입니다.
- **모든 파라미터 값은 필수값 입니다.**
- 택배의 배송 상태에 따라 배송 정보가 없을 수도 있습니다.

#### [URL]
**GET** /track2

#### [Parameter]
| 키          | 타입  | 필수  | 설명 |
| ----------- | ---- | --- | --- |
| parcelCode    | text | Y   | 송장번호 |
| parcelCompanyCode | text | Y   | 택배사 코드 **[<3. 택배사 코드 에서 확인 가능>](#3-택배사-코드)** |

#### [Example]
```
https://teparcel.lunasoft.co.kr/track2?parcelCode=653001352185&parcelCompanyCode=1
```
